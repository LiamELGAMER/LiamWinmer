# LiamWinmer Proton Layer

Mi propia capa de compatibilidad personalizada para Steam Play en la Steam Deck.

## Cómo instalar de forma manual:
1. Descarga estos archivos.
2. Crea una carpeta llamada `LiamWinmer` en `~/.steam/steam/compatibilitytools.d/`.
3. Pega estos archivos dentro.
4. Copia la carpeta `files` de tu instalación local de `GE-Proton11-1` dentro de esta carpeta.
5. Reinicia Steam y ¡listo!
